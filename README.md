## 📊 Understanding the Technology

This project uses modern methods to read memes. It examines the text inside the meme and the visuals in tandem. This process is called multimodal analysis. The software uses pre-trained models to compare the features against known sexist language patterns. This method ensures that the tool captures nuances that text-only scanners miss. Researchers find this helpful for tracking trends in social media data over long periods.
